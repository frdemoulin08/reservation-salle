# Profils utilisateur de l’application

- Utilisateur non connecté (visiteur)
- Utilisateur connecté (usager)
- Gestionnaire administratif (traitement des demandes)
- Administrateur métier (paramétrage de l’offre)
- Responsable pilotage (accès aux statistiques)
- Super administrateur (paramétrage de l’application)

Les administrateurs métier ont les droits du profil administratif, et pas l’inverse.
