# Profils utilisateur de l’application

- Utilisateur non connecté (visiteur)
- Utilisateur connecté (usager)
- Gestionnaire administratif (traitement des demandes) — `ROLE_APP_MANAGER`
- Administrateur métier (paramétrage de l’offre) — `ROLE_BUSINESS_ADMIN`
- Superviseur (pilotage, statistiques) — `ROLE_SUPERVISOR`
- Super administrateur (paramétrage de l’application) — `ROLE_SUPER_ADMIN`

Les rôles sont **cumulables** mais **non hiérarchiques** : un utilisateur peut recevoir plusieurs rôles si nécessaire.
